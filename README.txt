R files for permutation test with RT-PCR data

1. R_Code_for_Analysis_of_RTPCRdata.rmd
	- includes all functions, descriptions of those functions and an example used in the paper. 
	- R markdown file
	- functions are not printed on the knitted document

2. R_Code_for_Analysis_of_RTPCRdata.html
	- the knitted document

3. Simulation_functions.r
	- includes all additional functions made for the simulation study in the paper.
	- detailed descriptions are given in the markdown file.

4. Simulation.r
	- example running code for the simulation study in the paper.
